# WEED Auto-Claim Bot

Automated bot for claiming WEED token rewards on Solana.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment
```bash
cp .env.example .env
```

3. Run the bot
```bash
npm start
```
